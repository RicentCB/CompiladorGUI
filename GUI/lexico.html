Analizador Lexico

En esta sección podras realizar anlizadores lexicos con expresiones regulares, dichas expresiones estaran dadas por cualquiera caracter, excluyendo: '(', ')', '&', '|', '+', '*', '?'.
Ya que dichos simbolos son usados para hacer las operaciones con dichas expresiones:

  • + : Cerradura Positiva
  • + : Cerradura Estrella
  • & : Concatenacion
  • | : Pipe u operacion OR
  • ? : Opcional
  • (): Usados para albergar otra expresion

Si fuese necesario hacer uso de estos simbolos "exclusivos", solo es necesario antecederlos con una diagonal invertida '\'. Ademas las expresiones puede ser usadas con rangos, como: "A-Z", "a-z", "0-9". Se puede hacer uso de rangos contenidos en estos; por ejemplo: "C-M", "2-5".

Pero nunca mezclando rangos entre ellos como "a-K", "0-z", "A-3", etc...

Se tiene que asignar un token cada expresion regular. Con dichas expresion se podra analizar una cadena, y el resultado sera el los "tokens" asociados a dicha cadena, si no se correspondiera con alguna expresion regular el token devuelto sera = -1
