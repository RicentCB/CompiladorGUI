Interprete

En esta sección podras realizar la interpretacion de un codigo, dicho interprete acepta statements:

If - Else
do while
switch - case

Todas las condiciones necesitan estar entre "()" parentesis, si se coloca mas de una instruccion a seguir estas deberan estar entre "{}".

En el lado izquierdo observaras un ejemplo de que sentencias pueden ser ejecutadas. 

            a = 10
            var1 = 0
            b = a
            c = 0
            PRINT "Primer IF"
            IF ( a == 10 AND var1 < 1) { 
                a = a + 1 
                b = a
            }ELSE{
                PRINT "Condicion Else"
            }
            WHILE (a < 10){
                a = a + 1
                IF (c != a){
                    c = SIN(PI)
                    c = a
                } ELSE{
                    c = COS(PI)
                    c = a + 1
                }
                PRINT c
            }
            PRINT a + 100
